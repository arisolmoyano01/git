## HU-01: Agregar Productos

### Descripción

Como encargado de un kiosco, quiero disponer de una funcionalidad que me permita registrar productos dentro del sistema para simular la gestión del inventario y organizar la información de los artículos disponibles para la venta.

### Objetivo

Contar con una herramienta básica que represente el proceso de incorporación de productos dentro del sistema de gestión del kiosco.

### Actor Principal

* Encargado del kiosco.

### Flujo Principal

1. El usuario ejecuta la funcionalidad de agregar productos.
2. El sistema procesa la solicitud.
3. El sistema muestra un mensaje indicando que el producto fue agregado correctamente.

### Criterios de Aceptación

* Debe existir una función destinada al registro de productos.
* La función debe poder ejecutarse desde el archivo principal del sistema.
* El sistema debe informar al usuario que la operación fue realizada correctamente.
* La funcionalidad debe integrarse con el resto de módulos del sistema.

### Resultado Esperado

Al ejecutar el programa, el sistema muestra el mensaje:

"Producto agregado correctamente"

confirmando la ejecución de la funcionalidad.
